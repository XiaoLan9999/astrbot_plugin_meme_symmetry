# astrbot_plugin_meme_symmetry

一个给 AstrBot 用的图片 meme 插件。

## 功能

- `/对称 左`
- `/对称 右`

支持：

- 当前消息直接带图
- 引用回复里的图片
- 静态图
- GIF 动图

## 用法

### 方式 1：引用图片
先引用一张图，再发送：

```text
/对称 左
```

或

```text
/对称 右
```

### 方式 2：直接带图
发送图片并附带命令：

```text
/对称 左
```

## 规则说明

- `左`：保留左半边，镜像到右半边
- `右`：保留右半边，镜像到左半边

## 安装

把整个插件目录放到：

```text
AstrBot/data/plugins/astrbot_plugin_meme_symmetry
```

然后在 AstrBot WebUI 插件页里重载插件。

## 依赖

插件目录内已经提供 `requirements.txt`：

```text
Pillow>=10.0.0
```

## 备注

这个插件故意把“取图逻辑”和“图像处理逻辑”拆开了，后续你要继续加：

- `/翻转 横`
- `/翻转 竖`
- `/灰度`
- `/反相`
- `/旋转 90`

会比较方便。
